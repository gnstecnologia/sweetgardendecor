Onde abrir o site e o painel

Funciona em qualquer servidor estático por HTTP(S): GitHub Pages, Live Server / Go Live no VS Code, etc. Não precisa Vercel.

Onde abre Carrega site-data.json? Grava no Git / pasta do projeto sozinho?
GitHub Pages ou Go Live Sim Não — use Exportar ZIP e depois commit (ou copiar arquivos) no repositório.
Duplo clique no HTML (file://) Geralmente não Não

Edição na tela fica na memória do navegador. Este painel também guarda um rascunho local (textos, URLs e ordem) neste aparelho; fotos enviadas por arquivo nesta sessão só entram de verdade no projeto após Exportar ZIP e atualizar o repositório.
